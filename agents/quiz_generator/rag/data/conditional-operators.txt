УСЛОВНЫЕ ОПЕРАТОРЫ В ПРОГРАММИРОВАНИИ

================================================================================
1. ОСНОВНЫЕ ПОНЯТИЯ
================================================================================

1.1 ЧТО ТАКОЕ УСЛОВНЫЙ ОПЕРАТОР

Определение: Условный оператор - это конструкция языка программирования, которая позволяет выполнять различные блоки кода в зависимости от истинности условия.

Ключевые характеристики:
- Условие: логическое выражение, которое может быть истинным (true) или ложным (false)
- Ветвление: возможность выбора между несколькими путями выполнения программы
- Управление потоком: изменение последовательности выполнения кода
- Логика принятия решений: реализация различных сценариев выполнения
- Булевы значения: результат условия всегда 0 (false) или 1 (true) в C

Пример на C:
if (age >= 18) {
    printf("Вы совершеннолетний\n");
}

---

1.2 УСЛОВИЕ И ЛОГИЧЕСКИЕ ВЫРАЖЕНИЯ

Условие: Выражение, которое имеет логическое значение (истина или ложь)

Примеры условий:
x > 5         // сравнение: больше ли x, чем 5
y == 10       // сравнение: равен ли y 10
z != 0        // сравнение: не равен ли z нулю
a < b         // сравнение: меньше ли a, чем b
c >= 100      // сравнение: больше или равно ли c 100
d <= 50       // сравнение: меньше или равно ли d 50

Логическое значение: В C число 0 означает false (ложь), любое другое число означает true (истина)

---

1.3 ОПЕРАТОРЫ СРАВНЕНИЯ

Оператор | Название          | Пример    | Результат
---------|------------------|-----------|------------------
==       | Равно            | 5 == 5    | true (1)
!=       | Не равно         | 5 != 3    | true (1)
>        | Больше           | 10 > 5    | true (1)
<        | Меньше           | 3 < 10    | true (1)
>=       | Больше или равно | 5 >= 5    | true (1)
<=       | Меньше или равно | 3 <= 10   | true (1)

Пример:
int x = 10, y = 5;
if (x > y) {
    printf("x больше y\n");  // выведет, так как 10 > 5
}

---

1.4 ЛОГИЧЕСКИЕ ОПЕРАТОРЫ

Оператор | Название | Описание                           | Пример         | Результат
---------|----------|-----------------------------------|----------------|----------
&&       | И (AND)  | Истина если оба условия истины    | true && true   | true
&&       | И (AND)  | Ложь если хотя бы одно ложь      | true && false  | false
||       | ИЛИ (OR) | Истина если хотя бы одно истина  | true || false  | true
||       | ИЛИ (OR) | Ложь если оба условия ложь       | false || false | false
!        | НЕ (NOT) | Инвертирует логическое значение  | !true          | false
!        | НЕ (NOT) | Инвертирует логическое значение  | !false         | true

Примеры:
int age = 25, has_license = 1;
if (age >= 18 && has_license) {
    printf("Можете ездить\n");  // оба условия истины
}

int is_weekend = 1, is_raining = 0;
if (is_weekend || is_raining == 0) {
    printf("Идите гулять\n");  // хотя бы одно истина
}

int is_blocked = 1;
if (!is_blocked) {
    printf("Доступ разрешён\n");  // это ложь, поэтому не выведет
}

================================================================================
2. ОПЕРАТОР IF
================================================================================

2.1 ПРОСТОЙ IF (IF БЕЗ ELSE)

Синтаксис:
if (условие) {
    // код, который выполняется если условие истинно
}

Описание:
- Проверяет условие
- Если условие истинно (не равно 0), выполняет блок кода
- Если условие ложно (равно 0), пропускает блок кода
- После блока if программа продолжает обычное выполнение

Пример:
int temperature = 30;
if (temperature > 25) {
    printf("Жарко!\n");  // выведет, так как 30 > 25
}
printf("Проверка завершена\n");  // выведет в любом случае

Пример 2:
int score = 50;
if (score >= 100) {
    printf("Вы выиграли!\n");  // не выведет, так как 50 < 100
}
printf("Конец игры\n");  // выведет в любом случае

---

2.2 IF-ELSE (ВЫБОР ИЗ ДВУХ ВАРИАНТОВ)

Синтаксис:
if (условие) {
    // код если условие истинно
} else {
    // код если условие ложно
}

Описание:
- Проверяет условие
- Если условие истинно, выполняет первый блок
- Если условие ложно, выполняет второй блок (else)
- Один из двух блоков всегда выполнится
- После блока программа продолжает обычное выполнение

Пример:
int age = 15;
if (age >= 18) {
    printf("Вы взрослый\n");
} else {
    printf("Вы ребёнок\n");  // выведет это, так как 15 < 18
}

Пример 2:
int balance = 100;
int withdrawal = 50;
if (balance >= withdrawal) {
    balance = balance - withdrawal;
    printf("Снятие успешно. Баланс: %d\n", balance);  // выведет это
} else {
    printf("Недостаточно средств\n");
}

---

2.3 IF-ELSE IF-ELSE (МНОЖЕСТВЕННЫЙ ВЫБОР)

Синтаксис:
if (условие1) {
    // код если условие1 истинно
} else if (условие2) {
    // код если условие1 ложно, но условие2 истинно
} else if (условие3) {
    // код если условие1 и условие2 ложны, но условие3 истинно
} else {
    // код если все условия ложны
}

Описание:
- Проверяет условия последовательно
- Выполняет первое условие, которое окажется истинным
- После выполнения одного блока остальные пропускаются
- else блок в конце опционален (необязателен)
- Может быть неограниченное количество else if

Пример:
int grade = 75;
if (grade >= 90) {
    printf("Отличная оценка\n");
} else if (grade >= 80) {
    printf("Хорошая оценка\n");
} else if (grade >= 70) {
    printf("Удовлетворительная оценка\n");  // выведет это
} else if (grade >= 60) {
    printf("Проходная оценка\n");
} else {
    printf("Не проходная оценка\n");
}

Пример 2 (Определение времени суток):
int hour = 14;
if (hour >= 5 && hour < 12) {
    printf("Доброе утро\n");
} else if (hour >= 12 && hour < 17) {
    printf("Добрый день\n");  // выведет это
} else if (hour >= 17 && hour < 21) {
    printf("Добрый вечер\n");
} else {
    printf("Доброй ночи\n");
}

---

2.4 ВЛОЖЕННЫЕ IF

Синтаксис:
if (условие1) {
    if (условие2) {
        // код если оба условия истинны
    }
}

Описание:
- Один блок if находится внутри другого блока if
- Внутренний if проверяется только если внешний if истинен
- Можно вложить неограниченное количество if
- Полезна для сложной логики с несколькими условиями

Пример:
int age = 25;
int has_license = 1;
if (age >= 18) {
    printf("Вы взрослый\n");
    if (has_license) {
        printf("Вы можете водить машину\n");  // выведет оба сообщения
    } else {
        printf("Вам нужно получить права\n");
    }
} else {
    printf("Вы ребёнок\n");
}

Пример 2 (Проверка доступа):
int user_id = 5;
int is_admin = 1;
int is_active = 1;

if (user_id > 0) {
    if (is_active) {
        printf("Пользователь активен\n");
        if (is_admin) {
            printf("Это администратор\n");  // выведет все три сообщения
        }
    } else {
        printf("Пользователь неактивен\n");
    }
} else {
    printf("Неверный ID пользователя\n");
}

================================================================================
3. ОПЕРАТОР SWITCH-CASE
================================================================================

3.1 СИНТАКСИС И ИСПОЛЬЗОВАНИЕ SWITCH-CASE

Синтаксис:
switch (переменная) {
    case значение1:
        // код если переменная == значение1
        break;
    case значение2:
        // код если переменная == значение2
        break;
    case значение3:
        // код если переменная == значение3
        break;
    default:
        // код если переменная не равна ни одному значению
        break;
}

Описание:
- Проверяет значение одной переменной против нескольких значений
- Более читаемый альтернатива множественным if-else if
- Используется для сравнения с конкретными значениями (==)
- break; предотвращает "протекание" (fall-through) к следующим case
- default блок опционален, выполняется если ни один case не подошёл

Ключевые моменты:
- Переменная должна быть целочисленного типа (int, char, long и т.д.)
- Каждый case должен иметь уникальное значение
- break; критичен для правильной работы (без него выполнится следующий case)
- default блок выполняется если ничего не подошло

Пример:
int day = 3;
switch (day) {
    case 1:
        printf("Понедельник\n");
        break;
    case 2:
        printf("Вторник\n");
        break;
    case 3:
        printf("Среда\n");  // выведет это
        break;
    case 4:
        printf("Четверг\n");
        break;
    case 5:
        printf("Пятница\n");
        break;
    case 6:
        printf("Суббота\n");
        break;
    case 7:
        printf("Воскресенье\n");
        break;
    default:
        printf("Неверный день недели\n");
        break;
}

---

3.2 ОПЕРАТОР BREAK И FALL-THROUGH

break; - оператор, который прерывает выполнение switch блока

Описание:
- Выход из блока switch
- Предотвращает выполнение следующих case
- Обычно используется в конце каждого case

Пример БЕЗ break (Fall-through):
int choice = 1;
switch (choice) {
    case 1:
        printf("Один\n");
        // нет break!
    case 2:
        printf("Два\n");
        // нет break!
    case 3:
        printf("Три\n");
        break;
}
// Выведет: Один, Два, Три (все три сообщения!)

Пример С break (Правильно):
int choice = 1;
switch (choice) {
    case 1:
        printf("Один\n");
        break;
    case 2:
        printf("Два\n");
        break;
    case 3:
        printf("Три\n");
        break;
}
// Выведет: Один (только первое сообщение)

---

3.3 DEFAULT БЛОК

Синтаксис:
switch (переменная) {
    case значение1:
        // код
        break;
    case значение2:
        // код
        break;
    default:
        // код если ничего не подошло
        break;
}

Описание:
- Выполняется если переменная не равна ни одному case
- Похож на else в конце if-else if-else цепи
- Опционален (необязателен)
- Обычно размещается в конце

Пример:
char grade = 'X';
switch (grade) {
    case 'A':
        printf("Отлично\n");
        break;
    case 'B':
        printf("Хорошо\n");
        break;
    case 'C':
        printf("Удовлетворительно\n");
        break;
    default:
        printf("Неверная оценка\n");  // выведет это
        break;
}

---

3.4 MULTIPLE CASES С ОДНИМ БЛОКОМ

Синтаксис:
switch (переменная) {
    case значение1:
    case значение2:
    case значение3:
        // код для всех трёх значений
        break;
    default:
        break;
}

Описание:
- Несколько case могут выполнять один и тот же код
- Случаи перечисляются друг за другом без break
- Очень полезно для группировки похожих значений

Пример:
int day = 6;
switch (day) {
    case 1:
    case 2:
    case 3:
    case 4:
    case 5:
        printf("Рабочий день\n");
        break;
    case 6:
    case 7:
        printf("Выходной\n");  // выведет это
        break;
    default:
        printf("Неверный день\n");
        break;
}

================================================================================
4. ТЕРНАРНЫЙ ОПЕРАТОР (УСЛОВНОЕ ВЫРАЖЕНИЕ)
================================================================================

4.1 СИНТАКСИС ТЕРНАРНОГО ОПЕРАТОРА

Синтаксис:
условие ? выражение_если_истина : выражение_если_ложь

Описание:
- Однострочный способ написания простого if-else
- Возвращает значение (не оператор, а выражение!)
- Используется для присваивания или вычисления значения
- Три операнда (отсюда название "тернарный")
- Имеет самый низкий приоритет среди операторов

Ключевые моменты:
- Условие проверяется: если истина, вычисляется и возвращается первое выражение
- Если ложь, вычисляется и возвращается второе выражение
- Можно использовать в выражениях, присваиваниях, вызовах функций

Пример 1 (Простое присваивание):
int age = 20;
int status = (age >= 18) ? 1 : 0;  // status = 1, так как 20 >= 18
printf("Статус: %d\n", status);  // выведет: 1

Пример 2 (Присваивание строки):
int score = 85;
char *grade = (score >= 80) ? "Хорошо" : "Плохо";
printf("Оценка: %s\n", grade);  // выведет: Хорошо

Пример 3 (Вычисление в printf):
int x = 10;
printf("Число %d является %s\n", x, (x % 2 == 0) ? "чётным" : "нечётным");
// выведет: Число 10 является чётным

---

4.2 ВЛОЖЕННЫЕ ТЕРНАРНЫЕ ОПЕРАТОРЫ

Синтаксис:
условие1 ? выражение1 : (условие2 ? выражение2 : выражение3)

Описание:
- Несколько тернарных операторов могут быть вложены
- Работает как множественное if-else if-else
- Может быть сложна для чтения, используйте осторожно
- Обычно рекомендуется не вложивать более одного уровня

Пример:
int grade = 75;
char *result = (grade >= 80) ? "Отлично" : (grade >= 70) ? "Хорошо" : "Плохо";
printf("Результат: %s\n", result);  // выведет: Хорошо

Эквивалент с if-else if-else:
char *result;
if (grade >= 80) {
    result = "Отлично";
} else if (grade >= 70) {
    result = "Хорошо";
} else {
    result = "Плохо";
}

---

4.3 КОГДА ИСПОЛЬЗОВАТЬ ТЕРНАРНЫЙ ОПЕРАТОР

Хорошо использовать:
- Простые проверки (одно условие)
- Когда нужно присвоить значение переменной
- Для краткости кода в математических выражениях
- Внутри printf или других функций

Не рекомендуется использовать:
- Для сложной логики (используйте if-else)
- Когда нужно выполнить несколько операций в каждой ветви
- Когда более чем одно вложение (очень сложно читать)
- Когда это снижает читаемость кода

================================================================================
5. ЛОГИЧЕСКИЕ ОПЕРАТОРЫ (УГЛУБЛЕННОЕ ИЗУЧЕНИЕ)
================================================================================

5.1 ОПЕРАТОР && (И, ЛОГИЧЕСКОЕ И)

Синтаксис:
условие1 && условие2

Описание:
- Возвращает true (1) если ОБА условия истинны
- Возвращает false (0) если хотя бы одно условие ложно
- Используется для проверки нескольких условий одновременно
- Имеет "короткое замыкание": если первое условие ложно, второе не проверяется

Таблица истинности:
условие1 | условие2 | результат
---------|----------|----------
true     | true     | true
true     | false    | false
false    | true     | false
false    | false    | false

Пример:
int age = 25;
int has_license = 1;
if (age >= 18 && has_license) {
    printf("Можете ездить\n");  // выведет, оба условия истины
}

Пример 2 (Проверка диапазона):
int score = 75;
if (score >= 0 && score <= 100) {
    printf("Оценка в диапазоне\n");  // выведет
}

Важно - Короткое замыкание:
int x = 5;
if (x > 10 && x < 3) {  // первое условие ложно, второе не проверяется
    printf("Это не выведет\n");
}

---

5.2 ОПЕРАТОР || (ИЛИ, ЛОГИЧЕСКОЕ ИЛИ)

Синтаксис:
условие1 || условие2

Описание:
- Возвращает true (1) если ХОТЯ БЫ ОДНО условие истинно
- Возвращает false (0) если ОБА условия ложны
- Используется для проверки альтернативных условий
- Имеет "короткое замыкание": если первое условие истинно, второе не проверяется

Таблица истинности:
условие1 | условие2 | результат
---------|----------|----------
true     | true     | true
true     | false    | true
false    | true     | true
false    | false    | false

Пример:
int day = 6;
if (day == 6 || day == 7) {
    printf("Выходной\n");  // выведет, первое условие истины
}

Пример 2 (Проверка ошибки):
int error_code = -1;
if (error_code == -1 || error_code == 0) {
    printf("Ошибка или ноль\n");  // выведет
}

---

5.3 ОПЕРАТОР ! (НЕ, ЛОГИЧЕСКОЕ ОТРИЦАНИЕ)

Синтаксис:
!условие

Описание:
- Инвертирует логическое значение
- true становится false
- false становится true
- Используется для проверки "наоборот"
- Унарный оператор (работает с одним операндом)

Таблица истинности:
условие | результат
--------|----------
true    | false
false   | true

Пример:
int is_open = 0;
if (!is_open) {
    printf("Это закрыто\n");  // выведет, так как !0 = true
}

Пример 2 (Проверка NULL указателя):
int *ptr = NULL;
if (!ptr) {
    printf("Указатель NULL\n");  // выведет
}

Пример 3 (Отрицание условия):
int x = 10;
if (!(x > 20)) {
    printf("x не больше 20\n");  // выведет, так как 10 не > 20
}

---

5.4 КОМБИНИРОВАНИЕ ЛОГИЧЕСКИХ ОПЕРАТОРОВ

Описание:
- Логические операторы можно комбинировать для создания сложных условий
- Приоритет: ! выше, чем &&, && выше, чем ||
- Используйте скобки для ясности

Приоритет (от высшего к низшему):
1. ! (НЕ)
2. && (И)
3. || (ИЛИ)

Пример 1:
int age = 25;
int has_license = 1;
int is_sober = 1;
if (age >= 18 && has_license && is_sober) {
    printf("Можете ездить\n");  // выведет, все условия истины
}

Пример 2:
int temperature = 30;
int is_weekend = 1;
if (temperature > 25 || is_weekend) {
    printf("Идите на пляж\n");  // выведет, хотя бы одно истина
}

Пример 3 (Сложная логика):
int age = 17;
int has_parental_consent = 1;
if ((age >= 18) || (age >= 13 && has_parental_consent)) {
    printf("Можете использовать сервис\n");  // выведет
}

Пример 4 (С отрицанием):
int is_blocked = 0;
int is_active = 1;
if (!is_blocked && is_active) {
    printf("Доступ разрешён\n");  // выведет
}

================================================================================
6. ОСОБЫЕ СЛУЧАИ И ОШИБКИ
================================================================================

6.1 ОШИБКА: ПРИСВАИВАНИЕ ВМЕСТО СРАВНЕНИЯ

Ошибка - Использование = вместо ==:
int x = 10;
if (x = 5) {  // ошибка! это присваивание, не сравнение
    printf("x равен 10\n");
}
// x станет 5, условие истинно (так как 5 != 0), и выведет сообщение

Правильно - Использование ==:
int x = 10;
if (x == 5) {  // правильно, это сравнение
    printf("x равен 5\n");  // не выведет
}

Совет: Некоторые используют "reverse comparison" для предотвращения этой ошибки:
// if (5 == x) вместо if (x == 5)
// Если случайно напишете if (5 = x), компилятор выдаст ошибку

---

6.2 ОШИБКА: ЗАБЫВАНИЕ BREAK В SWITCH

Ошибка - Забыли break (fall-through):
int choice = 1;
switch (choice) {
    case 1:
        printf("Один\n");
        // забыли break!
    case 2:
        printf("Два\n");
        break;
}
// Выведет: Один, Два (два сообщения вместо одного)

Правильно:
int choice = 1;
switch (choice) {
    case 1:
        printf("Один\n");
        break;
    case 2:
        printf("Два\n");
        break;
}
// Выведет: Один (одно сообщение)

---

6.3 ОШИБКА: СРАВНЕНИЕ СТРОК С ==

Ошибка - Сравнение строк с ==:
char *name = "Alice";
if (name == "Alice") {  // ошибка! сравнивает адреса, не строки
    printf("Имя Alice\n");  // обычно не выведет
}

Правильно - Используйте strcmp():
#include <string.h>
char *name = "Alice";
if (strcmp(name, "Alice") == 0) {  // правильно
    printf("Имя Alice\n");  // выведет
}

---

6.4 ОШИБКА: ПУТАНИЦА С ЛОГИЧЕСКИМИ ОПЕРАТОРАМИ

Ошибка - Неправильное использование логических операторов:
int age = 25;
int has_license = 0;
if (age >= 18 || has_license) {  // считаете это ошибкой?
    printf("Может ездить\n");  // выведет, так как age >= 18 истинно
}

Это может быть логической ошибкой (ошибка в логике), а не в синтаксисе!
Правильно, если нужно И:
if (age >= 18 && has_license) {
    printf("Может ездить\n");  // не выведет, так как has_license = 0
}

7.4 ПРИМЕР 4: ВЫБОР ОПЕРАЦИИ (SWITCH-CASE)

#include <stdio.h>

int main() {
    char operation = '+';
    int a = 10, b = 5;
    int result;
    
    switch (operation) {
        case '+':
            result = a + b;
            printf("Сложение: %d\n", result);
            break;
        case '-':
            result = a - b;
            printf("Вычитание: %d\n", result);
            break;
        case '*':
            result = a * b;
            printf("Умножение: %d\n", result);
            break;
        case '/':
            if (b != 0) {
                result = a / b;
                printf("Деление: %d\n", result);
            } else {
                printf("Ошибка: деление на ноль\n");
            }
            break;
        default:
            printf("Неизвестная операция\n");
            break;
    }
    
    return 0;
}

Вывод: Сложение: 15

7.8 ПРИМЕР 8: ПРОВЕРКА ПРИВАТНОГО ЧИСЛА

#include <stdio.h>

int main() {
    int number = 17;
    int is_prime = 1;  // предполагаем, что простое
    
    if (number <= 1) {
        is_prime = 0;
    } else if (number == 2) {
        is_prime = 1;
    } else if (number % 2 == 0) {
        is_prime = 0;
    } else {
        for (int i = 3; i * i <= number; i += 2) {
            if (number % i == 0) {
                is_prime = 0;
                break;
            }
        }
    }
    
    printf("Число %d %s простое\n", number, is_prime ? "является" : "не является");
    
    return 0;
}

Вывод: Число 17 является простое

8. ТАБЛИЦА СРАВНЕНИЯ: IF VS SWITCH VS ТЕРНАРНЫЙ
================================================================================

Конструкция | Использование                  | Синтаксис             | Когда использовать
------------|--------------------------------|------------------------|------------------
if-else    | Проверка условий               | if (условие) { ... }   | Сложная логика, диапазоны
switch     | Множественный выбор одной переменной | switch (var) { ... } | Проверка конкретных значений
Тернарный  | Простой выбор двух значений    | условие ? знач1 : знач2 | Однострочные выражения

9. ЛУЧШИЕ ПРАКТИКИ
================================================================================

9.1 ИСПОЛЬЗОВАНИЕ ОПЕРАТОРОВ СРАВНЕНИЯ

- Всегда используйте == для сравнения, не = для присваивания
- Используйте скобки для ясности: if (a > 5 && b < 10)
- Для сравнения строк используйте strcmp(), не ==
- Помните о типах данных при сравнении (int, float, char)

---

9.2 ЛОГИЧЕСКИЕ ОПЕРАТОРЫ

- && имеет выше приоритет, чем ||
- Используйте скобки для ясности: if ((a && b) || c)
- Помните о "коротком замыкании" логических операторов
- Не смешивайте слишком много логических операторов в одном условии

---

9.3 СТРУКТУРА IF-ELSE

- Размещайте наиболее вероятные условия в начале
- Используйте else if для множественных условий, не вложенные if
- Избегайте глубокой вложенности (максимум 3 уровня)
- Используйте фигурные скобки даже для одной строки кода

---

9.4 ИСПОЛЬЗОВАНИЕ SWITCH

- Используйте switch для проверки конкретных значений
- Никогда не забывайте break после каждого case
- Используйте default для обработки неожиданных значений
- switch работает только с целочисленными типами и char

---

9.5 ЧИТАЕМОСТЬ КОДА

- Правильно форматируйте код (используйте отступы)
- Используйте осмысленные имена переменных
- Добавляйте комментарии для сложной логики
- Избегайте слишком сложных условий (разделите на несколько переменных)

10.1 ВОПРОСЫ НА ПОНИМАНИЕ

Вопрос 1: Что возвращает условие (5 > 3) в C?
Ответ: 1 (true), так как 5 действительно больше, чем 3

Вопрос 2: В чём разница между if-else и switch?
Ответ: if-else проверяет условия (>, <, >=, и т.д.), switch проверяет конкретные значения (==)

Вопрос 3: Что произойдёт, если забыть break в case блока switch?
Ответ: Выполнение "упадёт" на следующий case (fall-through), выполнив оба блока

Вопрос 4: Какой оператор имеет выше приоритет: && или ||?
Ответ: && (И) имеет выше приоритет, чем || (ИЛИ)

Вопрос 5: Что такое "короткое замыкание" в логических операторах?
Ответ: Если первый операнд && дает false, второй не проверяется; если первый || дает true, второй не проверяется

Вопрос 6: Какой результат тернарного оператора (10 > 5) ? 1 : 0?
Ответ: 1, так как 10 действительно больше, чем 5, возвращается первое значение

Вопрос 7: Что выведет следующий код?
int x = 5;
if (x = 10) {
    printf("Десять\n");
}
Ответ: "Десять" будет выведено, так как происходит присваивание (x = 10), а не сравнение, и 10 != 0 (истина)

Вопрос 8: Как проверить, находится ли число в диапазоне от 1 до 100?
Ответ: if (number >= 1 && number <= 100)

11. РЕЗЮМЕ КЛЮЧЕВЫХ КОНЦЕПЦИЙ
================================================================================

1. Условный оператор позволяет ветвить выполнение программы
2. Операторы сравнения (==, !=, >, <, >=, <=) возвращают 0 или 1
3. if выполняет блок если условие истинно
4. if-else выполняет один из двух блоков
5. if-else if-else проверяет множественные условия последовательно
6. switch проверяет переменную против конкретных значений
7. break критичен в switch для предотвращения fall-through
8. && (И) - оба условия должны быть истины
9. || (ИЛИ) - хотя бы одно условие должно быть истинно
10. ! (НЕ) инвертирует логическое значение
11. Тернарный оператор (условие ? знач1 : знач2) - укороченная форма if-else
12. Всегда используйте == для сравнения, не = для присваивания
13. Используйте скобки для ясности в сложных условиях
14. Предпочитайте if-else if-else вложенным if для множественных условий
15. Помните о "коротком замыкании" логических операторов
