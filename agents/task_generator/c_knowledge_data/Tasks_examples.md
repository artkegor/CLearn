**RAG 1 агент**

**Переменные и типы данных**

Простой уровень

1\. Объяви целочисленную переменную, присвой ей значение, выведи на
экран.

2\. Задай переменную тип float, вычисли квадрат заданного числа и
выведи результат.

3\. Введи с клавиатуры два целых числа, посчитай их сумму и разность,
выведи оба результата.

4\. Преобразуй введённое пользователем целое число в тип char и выведи
соответствующий символ.

5\. Объяви пару переменных: int и double. Сравни, как выводятся их
значения при присваивании дробного числа.

6\. Объяви переменную-логическую флаг (int isEven), проверь, чётное ли
число введено.

7\. Проинициализируй переменную unsigned int и введи отрицательное
число, выведи результат.

8\. Объяви переменную char, сохрани туда букву и напечатай её в
формате ASCII-кода.

9\. Напиши программу, которая принимает три значения типа int, находит
и выводит их среднее арифметическое.

10\. Преобразуй переменную типа double в int, выведи оба значения до
и после преобразования.

Средний уровень

1\. Напиши калькулятор: с клавиатуры вводятся два числа разного типа
(например, float и int), а также символ операции (+, -, \*, /),
программа вычисляет результат.

2\. Попроси пользователя ввести символ, выведи его предыдущий и
следующий символы в ASCII.

3\. Задай переменную long long, вычисли факториал числа (до 20!).

4\. Преобразуй дробное число, введённое с клавиатуры, к int с явным
кастом и объясни разницу с округлением.

5\. Введи с клавиатуры строку (массива char), посчитай количество букв
и цифр в строке.

6\. Используй три переменные (разного типа: int, float, double),
поочерёдно присваивай им значения друг друга, напиши, где теряются
данные.

7\. Сравни переменную unsigned int с обычным int: при вводе одного и
того же числа проверь, можно ли получить неожиданный результат при
вычитании.

8\. Проверь на переполнение: введи очень большое целое число в int и
вычисли сумму с другим большим числом, выведи результат.

9\. Допиши код, который "обменивает" значения двух переменных без
использования третьей.

10\. Создай две переменные типа float и double, присвой одну и ту же
дробную константу, сравни вывод с точностью до 10 знаков.

Сложный уровень

1\. Реализуй программу, определяющую минимальный и максимальный пределы
для каждого из базовых типов (int, float, char, double) через
limits.h и float.h.

2\. Считай число типа float, выведи его представление в памяти байтами
(используй указатель на char).

3\. Программа: пользователь вводит число, а ты определяешь --- может ли
оно быть безопасно преобразовано в тип short без потерь.

4\. Реализуй функцию, принимающую три вещественных числа и возвращающую
тип результата вычисления (например, сумма --- float, произведение ---
double, деление --- тоже число с определённым типом).

5\. Введи последовательность чисел разного типа, рассчитай сумму и
среднее для каждого типа (целые, дробные).

6\. Выведи на экран, сколько байт занимает каждый базовый тип и их
диапазоны.

7\. Напиши функцию преобразования из char\* в int через посимвольную
обработку (аналог atoi).

8\. Программа принимает ввод в виде строки, определяет, это целое,
вещественное или символ, и выводит тип.

9\. Проверь случай, когда выражение типа int x = 2147483647 + 1;
приводит к переполнению --- объясни, почему.

10\. Создай массив из 10 переменных разных базовых типов, наполни
случайными значениями, выведи все переменные через один printf с
форматированием.

**Условные операторы**

Простой уровень

1\. Программа определяет чётность введённого числа.

2\. Ввести число и вывести, является ли оно положительным, отрицательным
или нулём.

4\. Ввести два числа и вывести большее из них.

5\. Проверить, является ли год високосным.

6\. Определить, принадлежит ли число диапазону 10;100.

7\. Пользователь вводит символ --- определить, является ли он гласной.

8\. Проверить, совпадают ли два введённых числа.

9\. Ввести температуру и вывести сообщение: жарко, прохладно или холодно
(по градациям).

10\. Определить, входит ли введённая буква в слово "C programming".

Средний уровень

1\. Ввести три числа и найти наибольшее из них.

2\. Программа принимает три угла --- определить, могут ли они быть
углами треугольника.

3\. Ввести целое число --- определить по знаку и чётности, какое
сообщение вывести (например: чётное отрицательное и т.д.).

4\. По коду оценки (A, B, C, D, F) вывести словесную формулировку
оценки.

5\. Ввести свой возраст --- проверить доступ к разным сервисам (по
возрастным ограничениям, используя каскадные условия).

6\. Проверить, может ли прямоугольник с заданными сторонами быть
квадратом.

7\. Ввести три числа, определить, есть ли среди них одинаковые.

8\. Определить, в каком квартале года введён номер месяца (1--12).

9\. Ввести символ --- определить, цифра это, буква или другой символ.

10\. Программа вычисляет корни квадратного уравнения, выводит сообщение
о количестве и типе корней.

Сложный уровень

1\. Ввести дату и проверить, существует ли она (корректно ли число дней
для месяца и года).

2\. Программа на вход получает символ; если это буква, определить её
регистр и поменять его (A → a, z → Z).

3\. Ввести три стороны --- может ли существовать треугольник и какой он:
разносторонний, равнобедренный, равносторонний.

4\. Калькулятор с ветвлением: пользователь выбирает символ действия (+,
-, \*, /, %), программа ветвится по каждому кейсу с обработкой ошибок.

5\. Ввести дробное число, определить его знак и привести к целому типу с
правильным округлением (с учётом знака).

6\. На основе времени суток (целое значение 0--23) вывести, какое это
время: ночь, утро, день, вечер.

7\. Программа проверяет "магический квадрат" 3×3 --- все строки, столбцы
и диагонали дают одну сумму.

8\. Ввести дату, определить день недели (при фиксированной формуле или
готовой библиотеке).

9\. Проверить, являются ли три целых числа последовательными (например,
7, 8, 9).

10\. Оценщик баллов: ввести балл (0--100), вывести подробную градацию с
отличием, хорошо, удовлетворительно, неудовлетворительно, пересдача.

2\. Записать массив чисел в бинарный файл.

3\. Использовать fseek для позиционирования в файле.

4\. Тайминг: определить время выполнения операций с файлами.

5\. Модифицировать файл, заменяя символ.

6\. Чтение и запись блоками (buffered I/O).

7\. Обработка ошибок чтения/записи.

8\. Создать лог-файл и вести запись событий.

9\. Форматированный вывод с fprintf.

10\. Поиск подстроки в файле.

Сложный уровень

1\. Имплементация базы данных на основе бинарных файлов.

2\. Индексация данных в файле для быстрого доступа.

3\. Реализация файлового кэширования.

4\. Шифрование/дешифрование файла при записи/чтении.

5\. Управление несколькими открытыми файлами.

6\. Создание и использование временных файлов.

7\. Разделение файла на части и сбор их обратно.

8\. Реализация журналирования транзакций в файле.

9\. Использование mmap (если возможно) для доступа к файлу.

10\. Парсинг большого текста из файла с использованием потоков.

**Динамическая память**

Простой уровень

1\. Выделить память под массив из N целых с malloc.

2\. Освободить выделенную память с free.

3\. Инициализировать массив, выделенный динамически.

4\. Реализовать динамический массив строк.

5\. Использовать realloc для изменения размера массива.

6\. Выделить память под структуру и заполнить её поля.

7\. Проверка успешности выделения памяти.

8\. Создать двумерный массив с помощью динамической памяти.

9\. Написать функцию очистки выделенной памяти.

10\. Вывести все элементы динамического массива.

Средний уровень

1\. Имплементация динамического массива с автоматическим расширением
(вектор).

2\. Создать связный список с динамическим выделением памяти.

3\. Удаление элемента из динамического списка.

4\. Работа с несколькими указателями для управления памятью.

5\. Имплементация стека на динамической памяти.

6\. Написать функцию копирования динамического массива.

7\. Использование функции calloc и отличие от malloc.

8\. Реализация очереди с динамической памятью.

9\. Количество элементов в динамическом списке через структуру.

10\. Функция для перевыделения памяти с переносом старых данных.

Сложный уровень

1\. Имплементация двусвязного списка с динамической памятью.

2\. Сложное дерево с динамическими узлами (например, бинарное).

3\. Разработка простейшего аллокатора памяти.

4\. Управление памятью с помощью указателей на указатели.

5\. Отслеживание утечек памяти (с помощью вспомогательных структур).

6\. Реализация гибридных структур с динамическими и статическими
частями.

7\. Имплементация графа с динамическими смежными списками.

8\. Работа с данными неизвестного размера: строки без ограничения длины.

9\. Имитация сборщика мусора (простейший).

10\. Высокоэффективное сохранение и загрузка динамических структур.

**Препроцессор**

Простой уровень

1\. Использовать #define для объявления констант.

2\. Мини-программа с использованием #ifdef и #ifndef.

3\. Макрос для вычисления квадрата числа.

4\. Использование #include для подключения заголовочного файла.

5\. Комментарии препроцессора #error и #warning.

6\. Определить макрос с параметрами.

7\. Использовать #undef для удаления определения макроса.

8\. Использовать директиву #line.

9\. Написать макрос с тернарным оператором.

10\. Считать размер массива с помощью #define.

Средний уровень

1\. Определить макрос для обмена двух переменных.

2\. Использовать условную компиляцию для платформо-зависимого кода.

3\. Написать макрос для логирования с включением имени файла и строки.

4\. Использование #pragma для управления предупреждениями компилятора.

5\. Создать макрос с несколькими выражениями (блок в фигурных скобках).

6\. Использовать макросы для упрощения чтения кода с битовыми масками.

7\. Условная компиляция, зависящая от версии компилятора.

8\. Макросы для генерации функций с разными типами.

9\. Объявить макрос, который вызывает другую функцию.

10\. Реализация отладочного режима с помощью макросов.

Сложный уровень

1\. Макрос с рекурсивным вызовом (через выражения).

2\. Имплементация простого тестового фреймворка через препроцессор.

3\. Генерация кода с использованием конкатенации токенов ##.

4\. Макросы для создания синглтон-класса (в С++. в С аналог, управляющий
глобальной структурой).

5\. Расширенные макросы с вариантами поведения (в зависимости от
параметров).

6\. Анализ и защита от повторных включений файлов с #pragma once.

7\. Обработка компиляции с разными стандартами языка через препроцессор.

8\. Использование макросов для логирования ошибок с контекстом в больших
проектах.

9\. Автоматическое инкрементирование версий с помощью макросов.

10\. Метапрограммирование на С с использованием макросов для генерации
повторяющегося кода.
